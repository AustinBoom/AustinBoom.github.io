    • Compiler Construction.
      • Completing this class successfully on the first try was a great accomplishment for me. This class has a narrow pass rate at UVU and is a collective of everything you have learned in your degree. In this class I built a compiler from scratch, by myself, which compiles a language called kxi into target assembly code. Kxi is an object oriented language, similar to Java or C++. The target ASM code that is generated from the compiler is then fed into the virtual machine to be executed. The virtual machine is the project from the previous semester.
    • Advanced High-Performance Computer Architecture.
      • In this class we build an assembler and a virtual machine that runs the assembler, all from scratch. (Down to the byte code!) The virtual machine uses a RISC-based architecture, with 8 registers, jump instructions, move instructions, arithmetic instructions, logcal instructions, compare instructions, traps, directives, and three addressing modes: direct, immediate, and register modes. It has a fully functional stack, and therefore supports recursion, along with having a heap. This was my favorite class in college and where I matured as a programmer.
